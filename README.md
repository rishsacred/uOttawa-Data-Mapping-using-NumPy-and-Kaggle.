# uOttawa-Data-Mapping-using-NumPy-and-Kaggle.
This project builds a Machine Learning model to predict the presence of heart disease using clinical patient data.

The model is trained using a Random Forest Classifier and evaluates performance using accuracy, classification reports, and confusion matrices.

The dataset contains patient health indicators such as age, blood pressure, cholesterol, maximum heart rate, and more.


🛠️ Requirements
Install the required libraries:

pip install pandas numpy matplotlib seaborn scikit-learn

▶️ How to Run
Place heart.csv in the same folder as your Python script.

Run:
python your_script_name.py

Output:
Printed model performance in terminal
Saved PNG visualization files in project directory
